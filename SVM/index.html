🧠 什麼是 SVM？

支援向量機是一種試圖找到「最寬街道」(最大邊距) 來區分不同類別資料的演算法。

那些落在街道邊緣的關鍵資料點，就被稱為「支持向量」。

🔮 核函數技巧 (Kernel Trick)

當資料無法用直線分開時，SVM 可以將資料投影到更高維度，就像把一張紙捲起來剪開一樣。

RBF Kernel 就像是在資料周圍畫圈圈。

🎚️ C 值的影響

  • C 很大：對錯誤分類容忍度低，邊界較窄 (Hard Margin)。
  • C 很小：容許一些錯誤，邊界較寬 (Soft Margin)，較不易過度擬合。
